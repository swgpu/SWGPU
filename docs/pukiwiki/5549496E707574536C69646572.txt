* Class: UIInputSlider

A UI widget displaying a slider input.~
It emit 'E_VALUE_CHANGED' when value changed.~
- inherit from: UIWidget~
** Constructors
- ''new UIInputSlider''(): UIInputSlider~
** Methods
- ''onAction''(actionId: string): void~

~

~
- ''setStep''(step: number): void~
