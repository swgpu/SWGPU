box

  • box(x, y, z, radius, height, mx, my, mz, lift?, snapFloor?, snapFloorDistance?, marginBox?): ResBox

  • Returns a new move with smooth sliding along wall and floor for the given box. Infos are composed to a move vector, a wall collide flag and floor collide flag.

    Parameters

    • x: number

      The x position of the box center.

    • y: number

      The y position of the box center.

    • z: number

      The z position of the box center.

    • radius: number

      The radius of the box.

    • height: number

      The height of the box.

    • mx: number

      The movement in the x-axis.

    • my: number

      The movement in the y-axis.

    • mz: number

      The movement in the z-axis.

    • lift: number = 0.2

      The lift is used to elevate the virtual bounding box to let passing over little step or micro obstacles on the floor.

    • snapFloor: boolean = true

      Enable or disable floor snapping.

    • snapFloorDistance: number = 1

      Minimum distance to snap the floor.

    • marginBox: number = 0.1

      The margin is used to add more space to the virtual bounding box to collide wall collide earlier.

    Returns ResBox
